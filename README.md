## CrossPost (Multi Post)

Official site for the CrossPost desktop application (listed as "Multi Post" in the TikTok for Developers portal).

- Local-first desktop tool
- Uses only official platform APIs (TikTok Content Posting API + Login Kit, etc.)
- No bots, no scraping
